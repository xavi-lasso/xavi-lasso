Kelly Wearstler Headtrips

A collaboration for renowned LA based designer Kelly Wearstler. The project imagined an immersive web experience, where the user can explore beautiful spaces created by the KW team, and mint a unique NFT inspired by Kelly's Headtrip sculptures.

Process

Half of the project consisted on the development of the environments to run in Unity as a real-time immersive experience. For this, we generated a procedural setup that could process large amounts of geometry in order to optimize it by remeshing, laying out UV maps and cleaning up attributes.

Process

The other half was the Headtrip Generator, a highly detailed procedural system that could generate 10K+ iterations of photorrealistic interpretations of Kelly's striking sculputures.

Leveraging Houdini's PDG capabilities, we built a series of mini-generators that could produce infite variations of each garment decorating the sculputures, and scatter them in specific areas. The final process was a random material generator that would affect both the base and garments of each render.
